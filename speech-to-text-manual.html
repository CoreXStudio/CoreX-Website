Installation Guide

  1. Download the Application

    Download the Speech To Text application from the official CoreX website or use the link below.

  2. Locate the Installer

    Find the downloaded file in your Downloads folder.

  3. Run the Installer

    Double-click the installer file to begin installation.

  4. Security Confirmation

    If Windows asks whether you want to run the file, select "Run". If a "More info" link appears, click it, then choose "Run anyway" only if you are sure you downloaded the file from the official source.

Security Notes

⚠️ Important: Sometimes Windows Defender or other antivirus software may mistakenly identify new software as unsafe. This is called a false positive.

If Security Warnings Appear:

  1. Verify the Source

    Make sure you downloaded Speech to Text only from the official CoreX website.

  2. Windows SmartScreen Warning

    If you see a Windows SmartScreen or antivirus warning, choose "More Info" → "Run anyway" only if you are certain it's from our official site.

  3. Antivirus Blocking Installation

    If your antivirus blocks the installation, you can:

    • Look for an "Allow", "Keep", or "Restore" option in your antivirus notifications.
    • Add Speech to Text to your antivirus program's trusted list (instructions vary depending on your antivirus software).
  4. Still Unsure?

    If you are unsure, please contact our support team on Discord via our website.

⚠️ Security Reminder: Do not disable your antivirus entirely. This protects your system. Only adjust settings for Speech to Text if you are confident you have the genuine installer.

Getting Started

First-Time Setup

When you first launch the application, you'll need to complete these setup steps:

  1. Open the Application

    Launch the Speech to Text application from your desktop or start menu.

  2. Configure Microphone

    Add your microphone to your input settings to allow the software to work properly.

  3. Connect to VRChat

    Click on the "Connect to VRChat" button to establish a connection with your VRChat application.

  4. Start the Application

    Click on the "Start" button to begin the speech recognition process.

Using Speech to Text

Follow these simple steps to use the Speech to Text software:

  1. Start the Application

    Click the Start button to begin speech recognition.

  2. Speak Clearly

    Speak clearly into your microphone for best results.

  3. View Results

    Your words will appear in the text box in real-time.

  4. Stop Recording

    Click Stop to finish the speech recognition session.

  5. Save Your Work

    Save or copy your transcription as needed.
